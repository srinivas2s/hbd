# Birthday Website Project

A premium birthday website background design featuring a custom grid system and organic corner curves.

## Tech Stack

The following technologies were used to build this project:

- HTML5
- CSS3 (Vanilla)
- Google Fonts (Outfit)

## Project Structure

```text
hbd/
├── index.html    # Main HTML structure
├── style.css     # Design system, grid, and animations
└── README.md     # Project documentation
```

## Features

- Full-screen custom grid background with customizable size.
- Organic curved accents in the top-left and bottom-right corners.
- Subtle paper-fiber texture overlay for a premium feel.
- Responsive design that adapts to various screen sizes.
